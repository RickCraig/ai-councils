## Contributing

The AI Council Toolkit is open source and welcomes contributions. If you have
a suggestion, found an error, or want to add content:

- **Open an issue** to start a discussion or report a problem
- **Submit a pull request** if you would like to contribute content directly

See the [README](https://github.com/RickCraig/ai-council-toolkit) for
contribution guidelines and project setup instructions.
